# Victory Fit — Salle de sport à Metz

## Brand
- Colors: `#7A1B1F` (burgundy, primary), `#F29031` (orange, accent), `#EF6C3E` (deep orange). Gradient: `#F29031 → #7A1B1F`.
- On dark/colored bg → white logo. On light bg → black logo + orange accents.
- Fonts: Domus Titling (H1 bold, H2 semi-bold) / New Zen (H3 medium, body regular). Placeholders: Bebas Neue + Inter.

## Tech
- Astro static site, global CSS + scoped styles, AOS animations
- `npm run dev` / `npm run build` → `dist/`
- Pages: index, concept, coach, cours, coaching, planning, tarifs, contact, mentions-legales

## Structure
- `src/components/` `src/layouts/` `src/pages/` `src/styles/global.css`
- `public/assets/` — images, `assets/copywriting/*.md` — textes de référence
- `TASKS.md` — audit & tâches restantes
